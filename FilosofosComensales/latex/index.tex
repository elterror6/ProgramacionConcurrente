\hypertarget{index_autotoc_md0}{}\doxysection{Introducción}\label{index_autotoc_md0}
Este programa simula el problema de los filósofos comensales, donde varios filósofos alternan entre pensar y comer. Utiliza buzones para sincronizarse con los demás filósofos.\hypertarget{index_autotoc_md1}{}\doxysection{Estructura del programa}\label{index_autotoc_md1}
El programa se divide en dos partes\+:
\begin{DoxyItemize}
\item {\bfseries{\mbox{\hyperlink{manager_8c}{manager.\+c}}}}\+: Se encarga de crear los procesos de los filósofos y gestionar los buzones.
\item {\bfseries{\mbox{\hyperlink{filosofo_8c}{filosofo.\+c}}}}\+: Simula el comportamiento de un filósofo, alternando entre pensar y comer.
\end{DoxyItemize}
